How to use
×
Welcome to stringsnthings!

- Double click anywhere on the viewport (outside boxes) to create a new text box.
- Drag boxes by the drag handle (↔↑↓ symbol on top-left of each box).
- Resize boxes by dragging the corners.
- Use the connect toggle button (blue/green circle) to create connections between boxes.
- Use the font size buttons (▲ and ▼) to adjust text size inside each box.
- Delete boxes using the × button.
- Use "Mark Center" to save your current viewport center.
- Use "Go to Center" to return to your marked center.
- Toggle Edit Mode to enable/disable editing of boxes and controls.

Enjoy your creative flow!
