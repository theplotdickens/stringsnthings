How to Use stringsnthings

Double click anywhere on the canvas to create a new text box.
Drag the ✥ symbol on the top left of each box to move it.
Use the ▲ and ▼ buttons on each box header to increase or decrease the font size.
Click the blue circle to toggle connections between boxes.
Use the resize handles (corners) to resize the boxes.
Click × on the box header to delete a box.
Use the Undo Delete button to restore the last deleted box.
Mark Center saves a point to return to with Go to Center.
Toggle Edit Mode enables or disables editing.
All changes are saved automatically.
